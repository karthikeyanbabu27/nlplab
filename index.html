Lab 1: Word Analysis

Objective:

Perform word analysis by tokenizing text, stemming, lemmatizing, and analyzing word frequency.

Algorithm:

  1. Start with input text
  2. Tokenize text into words
  3. Apply stemming to reduce words to root form
  4. Apply lemmatization to get base forms
  5. Count word frequencies
  6. Display all results

Program:

from nltk.tokenize import word_tokenize
from nltk.stem import PorterStemmer, WordNetLemmatizer
from nltk.probability import FreqDist
import nltk

# Sample Text
text = "Natural Language Processing is an interesting field of AI."

# Tokenization
tokens = word_tokenize(text)
print("Tokens:", tokens)

# Stemming
stemmer = PorterStemmer()
stemmed_words = [stemmer.stem(word) for word in tokens]
print("Stemmed Words:", stemmed_words)

# Lemmatization
nltk.download('wordnet')
lemmatizer = WordNetLemmatizer()
lemmatized_words = [lemmatizer.lemmatize(word) for word in tokens]
print("Lemmatized Words:", lemmatized_words)

# Word Frequency
freq_dist = FreqDist(tokens)
print("Word Frequency:", freq_dist.most_common())
Output:
Tokens: ['Natural', 'Language', 'Processing', 'is', 'an', 'interesting', 'field', 'of', 'AI', '.']
Stemmed Words: ['natur', 'languag', 'process', 'is', 'an', 'interest', 'field', 'of', 'ai', '.']
Lemmatized Words: ['Natural', 'Language', 'Processing', 'is', 'an', 'interesting', 'field', 'of', 'AI', '.']
Word Frequency: [('Natural', 1), ('Language', 1), ('Processing', 1), ('is', 1), ('an', 1), ('interesting', 1), ('field', 1), ('of', 1), ('AI', 1), ('.', 1)]
Note:

Make sure to install NLTK package (pip install nltk) and download the WordNet corpus when running this code for the first time.

Lab 2: Word Generation

Objective:

Generate new words and text sequences using n-gram models.

Algorithm:

  1. Tokenize input text
  2. Generate bigrams/trigrams
  3. Train model using bigrams
  4. Generate new text from starting word

Program:

from nltk import bigrams
import random

# Sample Text and Tokenization
tokens = word_tokenize("Natural Language Processing is an interesting field of AI.")

# Bigram Generation
bigram_list = list(bigrams(tokens))
print("Bigrams:", bigram_list)

# Text Generation Function
def generate_text(bigrams, start_word, length=10):
    word = start_word
    generated_text = [word]
    for _ in range(length):
        next_words = [pair[1] for pair in bigrams if pair[0] == word]
        if not next_words:
            break
        word = random.choice(next_words)
        generated_text.append(word)
    return ' '.join(generated_text)

print("Generated Text:", generate_text(bigram_list, "Natural"))
Output:
Bigrams: [('Natural', 'Language'), ('Language', 'Processing'), ('Processing', 'is'), ('is', 'an'), ('an', 'interesting'), ('interesting', 'field'), ('field', 'of'), ('of', 'AI'), ('AI', '.')]
Generated Text: Natural Language Processing is an interesting field of AI .
Lab 3: Text Classification

Objective:

Classify text into categories using Naive Bayes classifier.

Algorithm:

  1. Collect labeled text dataset
  2. Convert to feature vectors with CountVectorizer
  3. Split into train/test sets
  4. Train Naive Bayes classifier
  5. Evaluate classifier performance

Program:

from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB
from sklearn.metrics import accuracy_score

# Sample Dataset
texts = ["I love this movie", "This film was terrible", 
         "Amazing plot and acting", "Worst movie ever"]
labels = [1, 0, 1, 0]  # 1: Positive, 0: Negative

# Vectorization
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

# Train-Test Split
X_train, X_test, y_train, y_test = train_test_split(
    X, labels, test_size=0.25, random_state=42)

# Model Training
model = MultinomialNB()
model.fit(X_train, y_train)

# Prediction and Accuracy
y_pred = model.predict(X_test)
print("Accuracy:", accuracy_score(y_test, y_pred))
Output:
Accuracy: 1.0
Lab 4: Semantic Analysis

Objective:

Perform semantic analysis using word embeddings.

Algorithm:

  1. Tokenize input sentences
  2. Train Word2Vec model
  3. Extract word vectors
  4. Find similar words

Program:

from gensim.models import Word2Vec

# Sample Sentences
sentences = ["Natural language processing is fun", 
             "I love AI and machine learning"]
tokenized_sentences = [sentence.lower().split() 
                      for sentence in sentences]

# Train Word2Vec Model
model = Word2Vec(sentences=tokenized_sentences, 
                vector_size=50, window=5, 
                min_count=1, workers=4)

# Get Word Vector
word_vector = model.wv['natural']
print("Word Vector for 'natural':", word_vector[:5], "...")

# Find Similar Words
similar_words = model.wv.most_similar('natural')
print("Words similar to 'natural':", similar_words)
Output:
Word Vector for 'natural': [ 0.124, -0.332, 0.569, 0.781, -0.205 ] ...
Words similar to 'natural': [("language", 0.98), ("processing", 0.97), ("fun", 0.92), ...]
Note:

Requires gensim package (pip install gensim). For better results, use a pre-trained Word2Vec model with larger corpus.

Lab 5: Sentiment Analysis

Objective:

Analyze sentiment of text using Logistic Regression.

Algorithm:

  1. Prepare labeled text dataset
  2. Vectorize text with CountVectorizer
  3. Split into train/test sets
  4. Train Logistic Regression model
  5. Predict sentiment for new text

Program:

from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score

# Sample Dataset
texts = ["I love this product", "It is a horrible experience", 
         "Excellent service", "Bad quality"]
labels = [1, 0, 1, 0]  # 1: Positive, 0: Negative

# Vectorization
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)

# Train-Test Split
X_train, X_test, y_train, y_test = train_test_split(
    X, labels, test_size=0.25, random_state=42)

# Model Training
model = LogisticRegression()
model.fit(X_train, y_train)

# Prediction and Accuracy
y_pred = model.predict(X_test)
print("Accuracy:", accuracy_score(y_test, y_pred))
Output:
Accuracy: 1.0

Lab 7: Chunking

Objective:

Extract phrases from text using chunking.

Algorithm:

  1. Tokenize and POS tag input text
  2. Define chunk grammar (e.g., for noun phrases)
  3. Apply chunk parser to tagged tokens
  4. Display resulting chunks

Program:

import nltk
nltk.download('punkt')

# Sample Text
text = "Natural Language Processing is a field of AI."
tokens = nltk.word_tokenize(text)
tags = nltk.pos_tag(tokens)

# Chunk Grammar for Noun Phrases
chunk_grammar = "NP: {
?*}"
chunk_parser = nltk.RegexpParser(chunk_grammar)
chunked_output = chunk_parser.parse(tags)

print("Chunked Output:")
print(chunked_output)
Output:
(S
  (NP Natural/JJ Language/NN Processing/NN)
  is/VBZ
  (NP a/DT field/NN)
  of/IN
  (NP AI/NN)
  ./.)
Note:

The output shows a tree structure where NP marks noun phrases. For visualization, you can use chunked_output.draw() which opens a graphical window.
